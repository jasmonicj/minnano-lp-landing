# みんなのLP屋さん プライバシーポリシー
**最終更新日：2026年3月13日**

---

MODALAVA株式会社（以下「当社」）は、「みんなのLP屋さん」（以下「本サービス」）において取得するお客様の個人情報の取り扱いについて、以下のとおり定めます。

---

## 1. 取得する個人情報

本サービスでは、以下の情報を取得します。

| 情報の種類 | 取得方法 | 取得タイミング |
|-----------|---------|--------------|
| LINEユーザーID | LINE Messaging API | LINE友達追加時 |
| 氏名・担当者名 | ヒアリングシート | 申込み時 |
| 事業者名 | ヒアリングシート | 申込み時 |
| メールアドレス | 申込みフォーム | 申込み時 |
| 電話番号 | 申込みフォーム | 申込み時（任意） |
| サイトURL | ヒアリングシート | 申込み時 |
| 決済情報 | Stripe（外部決済サービス） | 決済時 |
| LINEでのやり取り内容 | LINE Messaging API | サービス利用中 |
| 紹介者情報 | クライアントの申告 | 申込み時 |

---

## 2. 個人情報の利用目的

取得した個人情報は、以下の目的に限り利用します。

1. 本サービスの提供（LP制作・修正・保守）
2. お問い合わせへの対応
3. サービスに関するご連絡（障害情報・料金変更等）
4. 請求・決済処理
5. サービス品質の向上（AI修正精度の改善）
6. 紹介インセンティブの管理・支払い

---

## 3. 個人情報の第三者提供

当社は、以下の場合を除き、お客様の個人情報を第三者に提供しません。

- お客様の同意がある場合
- 法令に基づく場合
- 人の生命・身体・財産の保護のために必要な場合

---

## 4. 個人情報の委託

当社は、サービス提供のために以下の外部サービスに個人情報の処理を委託します。

| サービス | 用途 | 会社 |
|---------|------|------|
| GitHub | ソースコード・データの管理 | GitHub, Inc.（米国） |
| Vercel | ホスティング・Webhook処理 | Vercel, Inc.（米国） |
| Stripe | 決済処理 | Stripe, Inc.（米国） |
| LINE Messaging API | メッセージ送受信 | LINEヤフー株式会社（日本） |
| Anthropic Claude API | AI処理 | Anthropic, PBC（米国） |
| Google Analytics（GA4） | アクセス解析（オプション利用時のみ） | Google LLC（米国） |

各サービスの個人情報の取り扱いは、各社のプライバシーポリシーに従います。

上記の委託先のうち、GitHub / Vercel / Stripe / Anthropic / Google Analytics は米国に所在する外国事業者です。個人情報保護法第31条に基づき、これらの外国事業者への情報提供については、本サービスのご利用をもって同意いただいたものとみなします。各社の個人情報保護の取り組みについては各社のプライバシーポリシーをご確認ください。

---

## 5. 個人情報の安全管理

当社は、個人情報の漏洩・滅失・毀損を防止するため、以下の安全管理措置を講じます。

- GitHubリポジトリのアクセス権限管理（非公開リポジトリ）
- Vercel KVへのアクセス制限
- スタッフへの個人情報取り扱い教育

---

## 6. 個人情報の保存期間

| データ | 保存期間 |
|--------|---------|
| LINEやり取り履歴・memory.md | 契約終了後1年間 |

## 7. 個人情報の開示・訂正・削除

お客様は、当社が保有するご自身の個人情報について、開示・訂正・利用停止・削除を請求することができます。ご請求は下記お問い合わせ先までご連絡ください。本人確認を行った上で、合理的な期間内に対応します。

本サービスのLPはCookieを利用する場合があります。Cookieはブラウザの設定で無効にすることができますが、一部機能が利用できなくなる場合があります。
