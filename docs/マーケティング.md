## 議題1：LPは作るか？

### 背景
SNS広告 → LINE登録 の導線において、間にLPを挟むかどうか。

### 選択肢

#### A：LP あり
```
SNS広告 → LP（サービス説明・料金・実績） → LINE登録CTA
```
**メリット**
- サービスへの理解・信頼を醸成してから登録させられる
- SEOにも使える（中長期）
- 料金・範囲を事前に説明できるので問い合わせ品質が上がる
- 「30万円を切る」という数字を見せて納得させてから来てもらえる

**デメリット**
- 制作コストがかかる（自社LPなので自分たちで作れるが）
- 離脱ポイントが増える

#### B：LP なし（広告 → LINE直結）
```
SNS広告 → LINE登録
```
**メリット**
- 摩擦が少なく登録率が上がる可能性
- 素早く始められる
- LINEの中でサービス説明できる

**デメリット**
- 理解・納得度が低い状態で登録される → 「思ってたのと違う」が増えやすい
- 料金への抵抗が高まりやすい

### 推奨（たたき台）
→ **A：LP あり** を推奨。初期費用15万という決して安くない金額への納得感と、
「ドメイン・ホスティング込み」「LINE一言で修正」という強みを伝える場として機能させる。
LP自体は自社サービスで作れば実例にもなる。

### 決定事項 ✅
**A：LP あり**
SNS広告 → LP → LINE登録のフローで進める。自社サービスで作ることで実例にもなる。

- **A**：LINE登録直後に決済（Stripe決済リンク送付）

→ **A** を推奨。オペレーションをシンプルにするため。LP（議題1）で説明が完結しているなら登録直後決済でよい。

### 決定事項 ✅
**A：LINE登録直後に決済**
LINE登録直後にStripe決済リンクを自動送付。LPで説明が完結しているため、摩擦を最小化する。

## 議題10：外部営業・紹介インセンティブ設計

### 決定事項 ✅

**方針：紹介制度のみ。外部営業委託は当面なし。**

紹介者がCV（契約）まで自発的にフォローする動機になる設計にする。

**インセンティブ**
- LINE登録時：報酬なし
- 契約（初期費用入金）時：紹介者に ¥50,000

**紹介タグの運用フロー**
1. 新規LINE登録 → 自動返信「お名前と、紹介してくださった方のお名前を教えてください」
2. クライアントが返信 → memory.mdに `referrer: [紹介者名]` を記録
3. 契約・入金確認時 → 紹介者に手動で¥50,000を支払い

**この設計の利点**
- 紹介者がCV前に自発的にフォローする動機になる（お金にならないと困るので）
- LINE登録だけの粗悪リードが混入しない
- システム側の実装がシンプル（referrerをmemory.mdに記録するだけ）

## 議題11：CPA設計・コスト試算

### 決定事項 ✅

**LTV・粗利・許容CPA**

```
LTV（1年契約・標準プラン）
  初期 ¥150,000 + 月額 ¥10,000 × 12 = ¥270,000

粗利率：約90%（インフラ・API代のみ）
粗利額：約¥243,000

許容CPA：粗利の30%以内 = ¥70,000〜80,000
```

**チャネル別CPA目標**

| チャネル | 目標CPA | 備考 |
|---------|--------|------|
| Meta広告 | ¥30,000〜40,000 | フェーズ1でテスト |
| 紹介制度 | ¥50,000（固定） | CV時のみ発生 |
| 有機（SNS・口コミ） | ¥0 | 中長期で育てる |

**脱毛サロンのCPA設計を参考にした理由**

脱毛（セルフ）と構造が似ている：
- 粗利率が高い → CPAを積んでも元が取れる
- サブスク構造 → LTVで計算できる
- 初期投資が重い業態がCPAを高く設定できる理論と同じ

ただし脱毛と違い固定費がほぼゼロなため、CPAの許容幅はさらに広い。

## 議題14：スケール設計（0.5稼働×2人/セット）

### 決定事項 ✅

**方針：フルタイム1人ではなく、0.5稼働フリーランス×2人を1セットとして運用する。**

理由：
- リスク分散（1人が離脱しても即死しない）
- 属人化防止（2人が互いに引き継ぎ資料になる）
- 採用母数が広がる（「週20時間・月¥250,000」で子育て中・副業フリーランス・地方在住にリーチ）
- コストが同じでリスクが半分

1セット（0.5×2人）でクライアント120社を管理。
3セット（0.5×6人）で360社・月収¥600万〜800万を目指す。
