- [ ] 開示に備えて住所・電話番号を準備しておくこと

- [ ] 店長採用時：相手のインボイス登録番号を業務委託契約書に記載（✅ テンプレート対応済み）

#### 4. 下請法
**リスク：** フリーランスへの業務委託に下請法が適用される場合あり

**本サービスは法人が運営するため下請法が適用されます。**

下請法の主な義務：
- 発注時に書面（業務委託契約書）を交付すること ✅ テンプレート作成済み
- 発注内容・報酬・支払期日を書面で明示すること ✅ 契約書テンプレートに含む
- 支払期日は役務提供完了日から60日以内に設定すること
- 不当な減額・返品・買いたたき等の禁止

**対応策：**
- [ ] 業務委託契約書を必ず締結（✅ テンプレート作成済み）
- [ ] 支払いサイトを役務提供から60日以内に設定（月末締め翌月払い等 ✅ 問題なし）
- [ ] 発注内容変更時も書面で通知する運用を徹底

---

#### 5. 個人情報保護法（越境移転）
**リスク：** GitHub・Vercel・AnthropicはUSサーバー → 個人情報の国外移転に該当

**対応策：**
- [ ] プライバシーポリシーに外部委託先（US企業含む）を明示（✅ 作成済み）
- [ ] 「外国にある第三者への提供」として同意を取得するか、委託先の安全管理措置を確認
- 実務上は個人情報保護委員会のガイドラインに従い、委託先の安全管理措置を確認・記録しておくことを推奨

---

#### 6. 最低契約期間1年・中途解約違約金
**リスク：** 消費者保護の観点から過度な違約金は問題になる場合がある

**ただし：** 本サービスはBtoB（事業者向け）のため消費者契約法の適用外
- 個人事業主に対してはグレーゾーンの可能性あり
- 利用規約で「残存期間の月額保守費を違約金として申し受ける」は一般的な範囲内

**対応策：**
- [ ] 個人事業主向けには申込み時に違約金条件を口頭でも説明
- [ ] 利用規約の同意フローで違約金条件を必ず確認させる
