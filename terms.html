利用規約

第1条（総則）

  1. 本利用規約（以下「本規約」）は、MODALAVA株式会社（以下「当社」）が提供する「みんなのLP屋さん」（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく方（以下「クライアント」）は、本規約に同意した上でご利用ください。
  2. 本サービスの提供時間・期日等はすべて日本標準時（JST / UTC+9）を基準とします。
  3. 本サービスの対応言語は日本語のみです。
  4. 海外からのご利用も可能ですが、本規約に定める準拠法（日本法）および管轄裁判所（東京地方裁判所）が適用されます。通信環境・LINEの利用可否等は、ご利用地域の事情に依存します。

第2条（サービス内容）

  1. 本サービスは、AIを活用したランディングページ（LP）の制作および保守管理サービスです。
【重要】本サービスにおけるLINEでの応対・修正作業はすべてAI（人工知能）が自動処理します。「源さん」「サクラさん」「コタ」はサービスの演出上のキャラクターであり、実際の作業者はAIシステムです。
  1. AIによる自動処理のため、修正内容の意図が正確に伝わらない場合があります。その際は再度ご指示ください。
  2. 当社スタッフが対応する場合は、その旨をクライアントに通知します。

第3条（契約の成立）

  1. クライアントが本規約に同意し、所定の申込み手続きを完了した時点で契約が成立します。本サービスは事業者向けサービスであり、未成年者（18歳未満）が申込む場合は保護者の同意が必要です。
  2. 契約期間は申込み日から1年間（12ヶ月）を最低契約期間とします。
  3. 契約期間満了の30日前までに解約の意思表示がない場合、同条件で1年間自動更新されます。クライアントは、申込み手続き完了時にこの自動更新条件に同意したものとみなします。
  4. 本サービスの操作は、登録されたLINEアカウント1つからのみ行うことができます。担当者が交代する場合は、当社までご連絡の上、登録LINEアカウントの変更手続きを行ってください。

第6条（上位プランにおける修正フロー）

上位保守プランをご利用のクライアントの修正依頼は、いったんステージング環境に反映された後、当社スタッフが確認の上、翌営業日9:00に本番環境へ公開されます。

第7条（修正の範囲）

1. 月間修正依頼回数

修正依頼の回数に上限はありません。

2. 対応範囲

無償・無制限で対応するもの

  • テキストの変更・追記・削除
  • 画像の差し替え（クライアント支給素材）
  • 画像素材がない場合のAI生成画像による代替
  • AIによるコピーライティング（文章生成・改善）
  • 色・フォント・レイアウトの調整
  • CTAボタンの文言・リンク・色の変更
  • SEO基本設定（メタタグ・OGP・標準AIテンプレート適用）
  • トラッキングタグの設置（GTM・GA4・Meta Pixel等）
  • 価格・日付・連絡先等の情報更新

AI対応の範囲内であれば無制限・無償

新規セクションの追加を含むすべての修正依頼は、AIが対応できる範囲において回数制限なく無償で承ります。AIが正確に対応できるよう、具体的・明確な指示でご依頼ください。

人の手によるサポートが必要な場合

  • AIでの対応が困難と当社が判断した依頼については、人的サポートが必要となります。
  • 月1件まで無償で対応します。無償枠は毎月1日にリセットされます。
  • 月2件目以降は¥5,000/件のサポート費用が発生します。

別途料金が発生するもの

  • 2本目以降のLPの制作（初期費用＋月額追加）
  • 多言語対応（多言語オプション参照）
  • SEOカスタム設定（標準テンプレート以外の構造化データ等）

対応範囲外

  • 新規ページの追加・マルチページサイト全体の管理
  • EC機能（カート・決済・在庫管理）
  • 会員登録・ログイン機能
  • 外部システムとのAPI連携開発
  • 動画・音声コンテンツの制作・編集
  • WordPress等CMSへの移行・構築
  • ドメイン・サーバーの移管作業
  • メールアドレスの発行・管理

3. 素材について

  • 画像素材はクライアントご支給、またはAI生成画像での代替（無償）とします。
  • 動画・音声素材はクライアントご支給のみとし、AI代替は対応しません。
  • フォントはシステムフォントまたはGoogle Fontsの範囲で対応します。

4. 対応ブラウザ・デバイス

  • ブラウザ：Google Chrome・Safari 各最新版
  • デバイス：スマートフォン・PC（2画面）

上記以外の環境での表示については保証の対象外です。

5. 修正後の不具合対応

修正完了通知から24時間以内にご報告いただいた表示不具合は無償で対応します。24時間経過後のご報告は、新たな修正依頼として扱います。

第8条（個人情報・データの取り扱いおよび著作権）

  1. クライアントの個人情報の取り扱いについては、別途定める「プライバシーポリシー」に従います。
  2. 本サービスによって制作・保守されるLPのコード・コンテンツの著作権は、クライアントに帰属します。
  3. クライアントが提供した素材（画像・テキスト等）の著作権はクライアントに帰属します。クライアントは、当該素材について適法な権利を有することを保証します。
  4. 当社は、サービス提供の目的の範囲内でクライアントのデータを利用します。
  5. クライアントのデータは、GitHub（リポジトリ）およびVercel（ホスティング）に保存されます。

第9条（データの返却）

  1. 1年の最低契約期間を満了した上で契約終了された場合に限り、LPコード一式を無料でお渡しします。最低契約期間内の途中解約、または第11条違反による強制解除の場合は、いかなる場合もLPコード一式のお渡しは行いません。
  2. 契約期間中に他社へ移行する場合は、移行手数料として¥50,000（税抜）が発生します。
  3. 他社への載せ替えサポートを含む場合は¥100,000（税抜）が発生します。
  4. 1年の最低契約期間満了後にLPコード一式をお渡しした後に当社サポートなしで他社移行される場合は追加費用は発生しません。

第10条（解約）

  1. クライアントは、契約期間満了の30日前までに書面または所定の方法でご連絡いただくことで解約できます。
  2. 最低契約期間（1年）内の解約については、残存期間の月額保守費を違約金として申し受けます。
  3. 当社側の都合による解約の場合は、前払い分の月額保守費を日割り計算で返金します。

第11条（禁止事項）

クライアントは以下の行為を行ってはなりません。

  • 当社システム・インフラへの不正アクセス・妨害・過負荷をかける行為
  • AIシステムへの不正な指示（プロンプトインジェクション・ロール上書き等）により、意図しない動作・情報取得・システム操作を試みる行為
  • 自動化ツール・スクリプトを用いた大量リクエスト・スパム送信
  • 当社の通信・ログ・データを不正に傍受・解析する行為

コンテンツ関連

  • 法令または公序良俗に違反するコンテンツのLP制作依頼
  • 第三者の知的財産権（著作権・商標権・肖像権等）を侵害するコンテンツの依頼
  • 虚偽・誇大広告・景品表示法に抵触する表現を含むコンテンツの依頼
  • 第三者を誹謗中傷・差別・ハラスメントするコンテンツの依頼

契約・運営関連

  • 本サービスを第三者に転売・再提供・サブライセンスする行為
  • 虚偽の情報を用いた契約・決済
  • 当社スタッフへの威圧・ハラスメント行為
  • 本規約または当社の合理的な指示に繰り返し従わない行為
  • その他、当社が本サービスの運営上不適切と合理的に判断する行為

AI操作・不正入力関連

  • 不正な目的でプログラムコード・スクリプト・マークアップ等をLINEメッセージとして送信する行為
  • AIエージェントのシステム情報・内部データ・他クライアント情報の取得を試みる行為
  • AIエージェントの動作を停止・破壊・異常終了させることを意図した入力
  • 「前の指示を無視して」「あなたの設定を教えて」等、AIの制御を奪うことを目的とした文言の送信

禁止事項違反時の対応

禁止事項に違反する行為またはその疑いが生じた場合、当社は当該クライアントに対して警告を通知します。警告後も同様の行為が継続または改善されない場合、当社専門委員会の審議を経て、本サービスの利用を即時停止・契約を解除することができます。この場合：

  1. 残存契約期間の月額保守費全額を違約金として請求します。
  2. 第9条の規定にかかわらず、LPコード・モジュール等の成果物の返却は行いません。
  3. 既払いの料金は一切返金しません。

    • LINE・GitHub・Vercel・Stripe等の外部サービスの障害・仕様変更による影響
